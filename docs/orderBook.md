# orderBook API  获取深度

请求格式：GET /v1/market/spots/orderBook/<symbol>`

请求参数：

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|symbol |是  |string |币对名称：BTC_USDT   |

请求示例：

```bash
GET https://api.zssh.com/v1/market/spots/orderBook/<symbol>
```
